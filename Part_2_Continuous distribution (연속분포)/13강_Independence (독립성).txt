독립성 Independence

사건간의 독립성은 이 전에 보았다.
사건간의 독립성과 확률변수간의 독립성의 차이를 강조하기 위해 이메일 예시도 보았다.


확률변수의 독립성
Independence of r.v.s	X1, X2, ..., Xn

정의	X1, X2, ..., Xn is independence
	if P(X1<=x1, X2<=x2, ..., Xn<=xn)	= P(X1<=x1) * P(X2<=x2) * ... * P(Xn<=xn)
	for 'all x1, x2, ..., xn'
	-> 이 확률분포를 "joint CDF"라 함	-> 각각의 확률분포함수를 "marginal CDF"라 함

	이 확률변수의 독립성이
	사건간의 독립성보다 간단해 보이지?
	예를 들면 사건간의 독립성에서는 세사건의 독립성을 보려면
	3개가 다 겹치는 부분 뿐만 아니라 2개씩 겹치는 곳도 고려해야 했다.
	하지만 위의 확률변수의 독립성을 볼 때에는 독립변수 Xk들이 전부 겹치는 부분만 봤지.
	확률변수의 독립성은 'all x1, x2, ..., xn'에 대하여 정의되므로 사실상 무한한 식이 있는 것. (?)
	위에 것은 '일반적 정의'일 뿐
	사건간의 독립성의 확인은 '유한한 식'으로 이루어 지잖아. 여기에는 무한한 식으로 확인... 응 더 복잡


discrete case	이산분포의 독립성
이산분포의 경우 누적분포함수보다 확률질량함수를 다루기 더 쉬우므로 
위의 연속분포에서 CDF를 다룬 것과 다르게 확률질량함수PMF로 독립성을 확인

정의	P(X1=x1, X2=x2, ..., Xn=xn)	= P(X1=x1) * P(X2=x2) * ...* P(Xn=xn)
	-> 'joint PMF	'	-> 'marginal PMF'
	이산분포에선 PMF로 확률분포의 독립성 확인하는 것과,  CDF로 확인하는 것이 같다. (?)
직관적	"joint PMF의 사건들 중에서 몇 개의 사건이 일어날 걸 아는 것이 다른 사건의 실행여부와 관계가 없다"
이해

쌍으로(pairwise) 독립성 확인 v.s. 전체(full) 독립성 확인	... 후자가 더 일반적인 독립성
쌍으로 보면 한 확률변수와 다른 특정 확률변수의 독립성만 알 수 있다
전체 독립성은 어떠한 조합의 사건 실행 여부를 알아도 나머지에 대해 아무것도 모르는 것. 완전한 독립성.

pairwise independence 는 적용되지 않고
full indepedence 만 적용되는 예제를 보자
EX	X1, X2 ~Bern(1/2) i.i.d.	
	X1과 X2가 1/2의 베르누이 확률변수라고 보자 (두개의 동전 던지기), X3 = 	1, if X1=X2
									0, other
	These are pairwise independence, but not independece
		X1과 X2 독립
		X1과 X3 독립	; X1을 안다고 X3를 알 수 없다
		X2와 X3 독립	; X2를 안다고 X3를 알 수 없다
		X1과 X2와 X3 종속	; X1과 X2를 알면 X3는 정해지기 때문
	따라서 쌍으로 독립성은 완전한 독립성을 가지기에 충분하지 않다
